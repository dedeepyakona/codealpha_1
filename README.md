🚀 Data Redundancy Removal System
📌 Project Description

This project is a cloud-based Data Redundancy Removal System developed using Python and Flask. The system is designed to identify and prevent duplicate data entries in a database, ensuring data accuracy and efficiency.

It allows users to:

Add user details (Name, Email, Phone)
Automatically detect duplicate entries based on email
Store only unique data
View all stored records through a simple web interface
🎯 Objective

The main objective of this project is to:

Prevent redundant data storage
Maintain clean and efficient databases
Demonstrate basic cloud deployment and database handling
⚙️ Technologies Used
Python 🐍
Flask (Web Framework) 🌐
SQLite (Database) 💾
HTML & CSS (UI Design) 🎨
Cloud Deployment using PythonAnywhere ☁️
🔥 Features
✅ Add new user data
✅ Duplicate detection using email
✅ Prevents redundant entries
✅ Displays stored data in table format
✅ Simple and user-friendly UI
✅ Deployed on cloud for public access
